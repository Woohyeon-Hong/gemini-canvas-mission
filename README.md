## 🧩 프로젝트 개요

### 1. ChronoCoach
시간 기록 기반 자기관리 웹앱
- 활동별 집중 시간 기록
- 주간/월간 누적 통계 제공
- AI 기반 활동 패턴 분석 및 개선 제안

---

### 2. 우아한 야바위
실력 기반 난이도 조정 야바위 게임
- 승률 기반 속도 변화 시스템
- 전적 저장 및 실시간 승률 분석

---

### 3. Loop2Stream
Java Stream API 실전 학습 퀴즈 앱
- 난이도/문제 수 선택
- 코드 제출 및 자동 채점
- AI 기반 코드 리뷰 및 종합 리포트 제공

---

### 4. 스네이크 아레나
2인 대전 배틀로얄 스네이크 게임
- 10초 단위 맵 축소 시스템
- 속도 증폭 아이템
- 충돌 판정 기반 승부 구조

---

## 🤖 AI 사용

AI는 단순 보조 기능이 아니라 각 앱의 핵심 기능으로 설계되었습니다.

- ChronoCoach → 활동 데이터 분석 및 개선 리포트 생성
- Loop2Stream → 문제 생성, 코드 채점, 코드 리뷰 및 학습 리포트 제공
